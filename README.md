# Bank Customer Churn Prediction

This project applies supervised machine learning models to predict customer churn in a banking dataset.

## Models Used

- Perceptron (baseline)
- Decision Tree
- Decision Tree with Hyperparameter Tuning
- Random Forest

## Techniques

- Cross Validation
- Grid Search
- Feature Importance Analysis

## Dataset

Bank Customer Churn Dataset (Kaggle)

## Accuracy comparison:

Perceptron: 0.76
Decision Tree: 0.80
Optimized Decision Tree: 0.86
Random Forest: 0.86
